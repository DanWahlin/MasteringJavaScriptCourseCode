What is 'use strict'?

View the results in the debugging console (ctrl/cmd+shift+i)

The 'use strict' option is in ECMASCript 5 and does the following:
  • Places a program or function into a "strict" operating context
  • Catches common coding bloopers, throwing exceptions
  • Prevents or throws errors when "unsafe" actions are taken
  • Disables confusing or poorly thought out features
Note that this example generates a lot of errors (on purpose) to show 'use strict' in action.
