JavaScript Properties

View the results in the debugging console (ctrl/cmd+shift+i)

This example shows different techniques that can be used to create properties.

Properties can be defined in multiple ways: 

    var person = {
        firstName: 'John',
        lastName: 'Doe'
    }
    person.age = 24;
    person['street'] = '123 Main St.';
    Object.defineProperty(Object.prototype, 
                          'loginAttempts',
                          { enumerable: false, value: 0});


By using defineProperty() you have more control over how properties can be used. The following example shows how to disable writes: 
    Object.defineProperty(obj, 'myProp', {
    value: 15,
    writable: false
});
